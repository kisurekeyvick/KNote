grid布局 (1) grid布局只会影响 如上：container盒子与project盒子 (2) 父级属性 display: grid/line-grid 设置内容为grid布局或者内联grid布局 (3) grid-template-columns 设置子属性的列 grid-template-rows 设置子属性的行 (4) grid-template-columns/row: repeat(数量, 对应的每个行列元素的值 可以存在多个) (5) grid-template-columns/row: repeat(auto-fill, 值) 对行列，的自动填充，即：例如一共有10个盒子，如果一行最多能放满8个100px宽度的div盒子，那么剩余两个放在第二行 (6) fr关键字，grid-template-columns: 1fr 2fr; 代表的是后一列的宽度是前一列宽度的2倍 同时fr可以与绝对长度的单位结合使用 .container { display: grid; grid-template-columns: 150px 1fr 2fr; } (7) minmax(min, max) 取一个范围内的值,表示长度就在这个范围之中 grid-template-columns: 1fr 1fr minmax(100px, 1fr) (8) auto关键字 auto关键字表示由浏览器自己决定长度 grid-template-columns: 100px auto 100px (9) grid-row-gap用于设置行间距，grid-column-gap用于设置列间距 grid-gap属性是grid-column-gap和grid-row-gap的合并简写形式，语法如下：grid-gap: ; .container { grid-gap: 20px 20px; } 如果grid-gap省略了第二个值，浏览器认为第二个值等于第一个值 根据最新标准：grid-column-gap和grid-row-gap写成column-gap和row-gap grid-gap写成gap (10) justify-items 属性、align-items 属性、place-items 属性 .container { justify-items: start | end | center | stretch; align-items: start | end | center | stretch; } justify-items属性设置单元格内容的水平位置（左中右），align-items属性设置单元格内容的垂直位置（上中下）。 start：对齐单元格的起始边缘 end：对齐单元格的结束边缘 center：单元格内部居中 stretch：拉伸，占满单元格的整个宽度（默认值） (11) justify-content 属性、align-content 属性、place-content 属性 .container { justify-content: start | end | center | stretch | space-around | space-between | space-evenly; align-content: start | end | center | stretch | space-around | space-between | space-evenly; } justify-content属性是整个内容区域在容器里面的水平位置（左中右），align-content属性是整个内容区域的垂直位置（上中下） (12) grid-column-start、grid-column-end .one { // 左边框是第二根垂直网格线 grid-column-start: 2; // 右边框是第四根垂直网格线 grid-column-end: 4; // 上边框是第二根垂直网格线 grid-row-start: 2; // 下边框是第四根垂直网格线 grid-row-end: 4; } (13) grid-column 属性、grid-row 属性 grid-column属性是grid-column-start和grid-column-end的合并简写形式 grid-row属性是grid-row-start属性和grid-row-end的合并简写形式 .one { grid-column: 1 / 3; grid-row: 1 / 2; } // 等同于 .one { grid-column-start: 1; grid-column-end: 3; grid-row-start: 1; grid-row-end: 2; }
